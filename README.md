# 🌊 ABYSS

> **A tense, top-down survival horror set in the deep ocean.** 

Your submarine has crashed 8,400 meters below the surface. Your oxygen is rapidly depleting. To survive, you must rely on your sonar to navigate a pitch-black labyrinth, manage sparse air tanks, and evade terrifying, sound-sensitive creatures that patrol the deep. Balance the need to see with the need to hide across 5 challenging levels. Can you survive the dark?


## ✨ Features
*   **Sonar-Based Stealth:** You can't see the map unless you ping your sonar, but every ping runs the risk of drawing the creatures straight to you. 
*   **Resource Management:** Carefully manage your rapidly draining oxygen while searching for sparse air tanks. 
*   **Strategic Power-ups:** Find and utilize **Speed Boosts**, **Shields**, and **Mega Pings** to outmaneuver the horrors of the deep. 
*   **Atmospheric Audio:** A dynamic, heartbeat-driven soundtrack that escalates as creatures close in on your position. 

## 🎮 Controls
* **Keyboard:** 
  * `W A S D` or `Arrow Keys` to swim.
  * `SPACE` to emit a Sonar Ping.
  * `ESC` to Pause.
* **Gamepad / Controller:** (Fully Supported!)
  * `Left Stick` or `D-Pad` to swim.
  * `Cross (A)` or `R1 (RB)` to emit a Sonar Ping.
  * `Options (Start)` to Pause.

## 🚀 Running the Game Locally
This game is built with **Phaser 3** and compiled via **Vite**. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abolambA/ABYSS.git
   cd ABYSS
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The game will be available at `http://localhost:5173/`.*

4. **Build for production:**
   ```bash
   npm run build
   ```
   *Static files will be generated in the `dist/` directory ready to be uploaded to itch.io or your web host.*
